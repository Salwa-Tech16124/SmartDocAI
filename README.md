## ✨ Enchanted Features

### 📝 Document Boutique (Agentic File Processing)
Just upload a document and tell the AI sprite what to do! It will autonomously route your file to the correct spell:
*   **PDF to DOCX:** Elegantly convert locked PDFs into editable Word documents.
*   **DOCX to PDF:** Freeze your documents into beautiful PDFs.
*   **Neural Translation:** Translate your `.docx` files to over 50 languages flawlessly.
*   **Magical Summarization:** Receive executive summaries from 100-page documents instantly.
*   **Compression Spells:** Shrink your large PDFs without losing quality.

### 👗 Career Stylist (ATS Analyzer)
Receive deeply analytical, recruiter-grade insights to refine your career profile. 
*   **ATS Scoring:** See exactly how well your resume matches the job description.
*   **Keyword Spells:** Identify missing skills required to bypass standard ATS filters.
*   **Formatting Polish:** Learn how to structure your profile to stand out beautifully.

## 🌸 Installation & Setup

Want to bring this magic to your own machine? Just follow these simple steps!

```bash
# 1. Clone the enchanted repository
git clone https://github.com/Salwa-Tech16124/SmartDocAI.git
cd SmartDocAI

# 2. Awaken your virtual environment
python -m venv .venv
.venv\Scripts\activate

# 3. Install the magic potions (dependencies)
pip install -r requirements.txt

# 4. Cast the final spell!
streamlit run app.py
```
