# SmartDoc AI 📄✨

SmartDoc AI is a powerful Streamlit-based web app that lets you:
- Convert Word to PDF and vice versa
- Translate DOCX files to any language
- Apply font styling
- Compress PDF files
- Hear a notification sound on success 🎵

## Features
✅ Word ↔ PDF Conversion  
✅ PDF Compression (Low/Medium/High)  
✅ AI-Based Translation (with styling)  
✅ Success Sound  
✅ Beautiful UI

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
